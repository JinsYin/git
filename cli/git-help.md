# git help

显示 Git 及其命令行的帮助信息（手册页）。

## 用法

```sh
git help [-a|--all [--[no-]verbose]] [-g|--guide]
            [-i|--info|-m|--man|-w|--web] [COMMAND|GUIDE]
```

## 其他

```sh
$ git help <verb>   # git help branch
$ git <verb> --help # git branch --help
$ git <verb> -h     # git branch -h
$ man git-<verb>    # man git-branch
```
