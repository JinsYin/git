# 二进制文件 & 文本文件

| 类型       | 描述                                    | 示例                                                                |
| ---------- | --------------------------------------- | ------------------------------------------------------------------- |
| 文本文件   | 字符编码（`UTF-8`、`ASCII`、`UNICODE`） | 源程序文件(`.py`、`.java` 等)、`.md`、`.rst`、`.txt`、`.csv`、`.sh` |
| 二进制文件 | 自定义编码                              | `.docx`/`.pptx`/`.xlsx`/`.exe`                                      |
