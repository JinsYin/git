# Github Pages 站点

## 数据源

| 面向 | 域名                             | 数据源（Pages 分支）                                                            |
| ---- | -------------------------------- | ------------------------------------------------------------------------------- |
| 个人 | `username.github.io`             | * `master` 分支                                                                 |
| 组织 | `orgname.github.io`              | * `master` 分支                                                                 |
| 项目 | `username.github.io/projectname` | 1. `gh-pages` 分支 <br> 2. `master` 分支 <br> 3. `master` 分支下的 `docs/` 目录 |
