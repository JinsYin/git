# Hub

## 安装

```sh
# macOS
$ brew install hub

# 验证
$ hub version
```

## 别名

```sh
# zsh
$ echo 'eval "$(hub alias -s)"' >> ~/.zshrc
```

## 示例

```sh
$ hub clone rtomayko/tilt # git clone git://github.com/rtomayko/tilt.git
```
