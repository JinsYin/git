# Git 引用

| 引用         | 路径                                              |
| ------------ | ------------------------------------------------- |
| 本地分支     | `.git/refs/heads/<local_branch_name>`             |
| 远程追踪分支 | `.git/refs/remotes/<origin>/<remote_branch_name>` |
| stash        | `.git/refs/stash`                                 |
| tag          | `.git/refs/tags/<tag_name>`                       |
